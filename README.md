# Marketing ROI Analysis via Simple Linear Regression

A data-driven analytics project designed to evaluate marketing budget allocation across media channels (TV, Radio, Social Media) and accurately predict Return on Investment (ROI).

---

## 📖 Project Overview
This project demonstrates how to apply fundamental statistical modeling to real-world business data. By analyzing past advertising expenditures, the model isolates which marketing channels yield the highest return, allowing organizations to optimize their future budget allocation.

> **Key Takeaway:** This repository serves as a practical, step-by-step guide to executing exploratory data analysis (EDA), validating statistical assumptions, and implementing regression modeling in Python.

---

*   **Data Manipulation:** `pandas` & `numpy`

Robust Data Loading (Object-Oriented & Safe)To ensure the pipeline doesn't crash from missing files, data loading is handled via a structured class with explicit error blocks:Pythonimport pandas as pd
import sys

class MarketingDataLoader:
    def __init__(self, file_path: str):
        self.file_path = file_path

    def load_data(self) -> pd.DataFrame:
        """Loads dataset snippets securely with exception handling."""
        try:
            df = pd.read_csv(self.file_path)
            print(f"📦 Successfully loaded dataset with {df.shape[0]} rows.")
            return df
        except FileNotFoundError:
            print(f"❌ Error: The file at {self.file_path} was not found.")
            sys.exit(1)
        except Exception as e:
            print(f"❌ An unexpected error occurred: {e}")
            sys.exit(1)
